# PRD: TubeNote

## 목표
유튜브 링크 하나를 던지면 영상의 핵심 내용을 깔끔한 마크다운 문서로 정리해주는 1인용 요약 서비스.

## 사용자
- 본인 1인 (개인용 도구)
- 학습/리서치 영상을 빠르게 훑고 노션에 아카이브하려는 사용 맥락

## 핵심 기능
1. 유튜브 링크 입력 → 영상 콘텐츠 분석 (자막/오디오 기반)
2. 분석한 내용을 구조화된 요약으로 변환 (목차, 핵심 포인트, 인용 가능한 문장 단위)
3. 결과를 마크다운(.md) 파일로 출력 — 다운로드 또는 로컬 저장
4. (선택) 출력된 마크다운을 본인의 노션 워크스페이스에 자동 페이지로 정리

## MVP 제외 사항
- 다국어 자막 자동 번역 (원본 언어 그대로 요약)
- 재생목록/채널 단위 일괄 요약 (링크 1개당 영상 1개)
- 영상 내 이미지·슬라이드 캡처 (텍스트 요약만)
- 사용자 계정/권한/공유 기능 (혼자 쓰는 도구라 불필요)
